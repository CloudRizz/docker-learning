## ▶️ How to Run

Most projects in this repository follow a standard Docker workflow:

### Build an image
```bash
docker build -t app-name .
````

### Run a container

```bash
docker run -p 5000:5000 app-name
```

### Run multi-container applications

```bash
docker-compose up --build
```
